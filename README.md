# RepoViz
RepoViz - Visualize any GitHub repository like never before.

---

## Features

- Repository health score
- Star and fork growth trends
- Contributor activity overview
- Issue response time analysis
- Outdated dependency detection
- One-click shareable report

---

## Download

Download the ZIP file, extract all contents to the same folder, then double-click RepoViz.exe to run.
